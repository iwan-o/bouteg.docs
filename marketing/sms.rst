Envoie groupé de Sms 
====================

Bouteg vous fait bénéficier d'un outil simple, rapide et performant pour envoyer des SMS groupés.

Cette fonction vous permet d'adresser à vos utilisateurs de votre base clients, elle est completement compatible avec les exigences de CNIL et qui respecte la loi RGPD.

Créer une nouvelle campagne SMS
--------------------------------

A partir du menu latéral cliquez sur la rubrique « SMS » puis cliquez sur le bouton « Créer » en haut à droite.

#. Rédiger votre SMS dans dans la zone « message »
#. Cliquez sur le bouton |image btn create| 
#. Puis cliquer sur le bouton prévisualiser pour passer à l'étape suivante

Filtrer les destinataires et prévisualiser
-------------------------------------------

Sur cette page vous pouvez choisir qui de vos clients recevras cet SMS 

Sur cette page vous pouvez corrigé le contenu et l'objet de votre e-mail.
Mais aussi vous envoyer un test en cliquant sur le bouton " M'envoyer un test". Ainsi vous receverz l'e-mail directement sur votre boite au lettre personnelle. 

.. image:: /_static/images/marketing/filtrer-les-images.png
    :alt: Filtrer les destinataires
    :class: img-capture-bordered

Vous pouver également prévisualiser votre message tel qu'il sera délivré a votre client 

.. image:: /_static/images/marketing/previsualisation-message.png
    :alt: Prévisualiser le SMS
    :class: img-capture-bordered

.. note::
    * Le nom de l'expéditeur sera "Bouteg".
    * La mention "STOP" est une mention légale qui est ajoutée a la fin de chaque SMS et qui permet a votre client de se désinscrire de votre liste de diffusion s'il le souhaitent. 

ll ne vous reste plus qu'a appuyez sur le bouton "envoyez" pour valider l'envoi de votre campagne SMS
